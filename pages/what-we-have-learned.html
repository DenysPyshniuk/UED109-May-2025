What We Have Learned

Our journey through web development fundamentals

Throughout this comprehensive course, we have explored the fundamental principles of web development, from basic HTML structure to advanced CSS techniques and content management systems. Our learning journey has equipped us with the skills needed to create modern, responsive websites.

Foundational Web Development

  • HTML Structure: Learned to build basic webpage structures using semantic elements, boilerplate, and development tools.
  • CSS Fundamentals: Gained proficiency in applying basic styling properties like colors, fonts, sizes, and spacing. Explored how to add CSS to HTML and understood the cascade.
  • Responsive Design: Introduced to principles of responsive design using media queries to adapt layouts for various screen sizes.

Advanced CSS Layout Techniques

  • CSS Display Properties: Deepened understanding of `display` properties, including `block` vs. `inline` elements.
  • CSS Box Model: Mastered the concept of the CSS Box Model, crucial for element sizing and spacing.
  • Flexbox: Explored Flexbox for efficient and flexible one-dimensional layout design.
  • CSS Grid: Learned CSS Grid for powerful two-dimensional layout systems.

Content Management Systems (CMS) - WordPress

  • Introduction to WordPress: Gained foundational knowledge of WordPress as a leading CMS.
  • Posts vs. Pages: Understood the fundamental differences and use cases for WordPress posts and pages.
  • Full Site Editing vs. Page Editing: Explored modern WordPress editing paradigms, including Full Site Editing and traditional Page Editing.
  • Plugins: Learned about the role and management of WordPress plugins to extend site functionality.

Development Workflow & Project Management

  • Version Control with Git & GitHub: Practiced using Git for version control and collaborating on projects via GitHub (cloning, branching, committing, pushing, pull requests).
  • Design Principles: Understood the importance of white space, cognitive load, and visual hierarchy in UI/UX design.
  • Project Planning: Engaged in creating website mockups and style guides.
  • AI Tools: Introduced to AI image generation tools like ImageFX.

Practical Application & Assessment

  • Class Activities & Labs: Applied theoretical knowledge through hands-on HTML and CSS labs, including semantic HTML and advanced CSS layouts.
  • Knowledge Tests: Consolidated learning through various knowledge tests covering HTML, CSS, and WordPress.
  • Midterm & Final Projects: Prepared for and completed a Midterm Exam and worked towards a comprehensive Final Project, integrating all learned skills.
