Tools & Platforms

Discover the tools and platforms we used during the course

In this course, students will create a responsive website using WordPress. Furthermore, students will gain expertise in translating design concepts into functional web applications, focusing on the front-end development aspect. They will acquire fundamental skills in website creation using modern technologies and best practices.

Tools and platforms students will work with

HTML

HyperText Markup Language is the standard language used to create and structure content on the web. It defines the layout and elements of a webpage, such as headings, paragraphs, links, and images.

CSS

Cascading Style Sheet is used to style and design HTML content. It controls colors, fonts, spacing, and layout, helping websites look visually appealing and user-friendly.

Visual Studio Code

Lightweight and powerful code editor developed by Microsoft. It supports many programming languages and features like syntax highlighting, debugging, and extensions, making it ideal for web development.

WordPress

Popular content management system (CMS) that allows users to build and manage websites easily. It offers customizable themes, plugins, and a user-friendly interface, perfect for blogs, portfolios, and business sites.
